# -QuizMaster

This project is a responsive and interactive Frontend Development Quiz Application designed using HTML, CSS, and JavaScript. The interface begins with a clean, centered start page that introduces the quiz, including total questions, marking scheme, and covered topics such as HTML, CSS, and JavaScript 

PRO

. The UI layout is structured inside a styled container featuring smooth rounded edges, a subtle shadow, and a dark-overlay background image that enhances the overall appearance. Custom CSS provides button styling, hover effects, spacing, alignment, and color themes to create an engaging user experience 

PRO

.

The core functionality is powered by JavaScript, where a set of 50 beginner-level MCQs is stored. Each time the quiz begins, the script randomly selects 15 unique questions to ensure a fresh attempt on every run 

PRO

. Users can navigate between questions using Previous and Next buttons, with their selections saved automatically. After completion, the application displays a detailed result page showing the score along with a complete answer review, highlighting correct and incorrect responses.

Overall, this project demonstrates strong understanding of DOM manipulation, event handling, dynamic content rendering, and responsive frontend design.
